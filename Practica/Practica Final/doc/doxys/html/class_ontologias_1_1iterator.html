Obtiene las temáticas a las que pertenece la palabra final que apunta el iterador.

Returns
una lista conteniendo todas las temáticas (con sus sinonimos). Asi cada elemento de la lista contiene una temática. Esta tematica se almacena en un conjunto (set<string>) donde se mantiene todos los sinónimos de esa temática incluida ella misma
