template<class T>
class VD< T >

T.D.A. Vector Racional.

Una instancia v del tipo de datos abstracto Vector Dinamico es un objeto de tipo vector dinamico, compuestos por un puntero a template, dos enteros que representan, respectivamente, el vector que contendrá los datos, el numero de elementos almacenados y el numero de elementos reservados.
